Evertales

Evertales est un jeu de rôle d’action en deux dimensions avec une vue de haut. Dans ce jeu, le joueur est plongé dans un univers médiéval fantastique avec l’objectif d’éliminer des monstres dans le but de devenir de plus en plus puissant. Pour développer notre jeu, nous avons utilisé Unity et nous nous sommes inspirés de Chronicon ainsi que de d'autres jeux dans le même genre tels que Diablo et Path of Exile.

Pour progresser dans son aventure, le joueur aura accès à une multitude d’habiletés (5 en ce moment) pour détruire les antagonistes qui obstruent son chemin. De plus, il dispose d’une vaste sélection d’équipements tous différents les uns des autres avec lesquels il peut équiper son personnage afin d’améliorer ses attributs.

Télécharger

Pour essayer la démo de notre jeu, cliquez sur le bouton puis téléchargez le zip qui se trouve à ce lien. Ensuite, il suffit de décompresser le fichier puis de démarrer l'éxécutable EverTales.exe.
